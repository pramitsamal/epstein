13.3.6 Procedure Nodes

There are nodes representing various sorts of procedures; these are kinds of ProcedureNode,
e.g.

e SchemaNode, indicating any procedure

e GroundedSchemaNode, indicating any procedure associated in the system with a Combo
program or C++ function allowing the procedure to be executed

e PredicateNode, indicating any predicate that associates a list of arguments with an output
truth value

e GroundedPredicateNode, indicating a predicate associated in the system with a Combo
program or C++ function allowing the predicateâ€™s truth value to be evaluated on a given
specific list of arguments

odes. We have seen many examples of EvaluationLinks in the above. Example ExecutionLinks
